## About
This is the personal portfolio website for **Aryan** — a motivated Frontend Developer (Intern) focused on building clean, responsive user interfaces using HTML, CSS and JavaScript. Currently learning React.js and working on production-quality apps.

The site is a single-page, responsive, and lightweight portfolio designed to:
- Showcase projects (live links + repo links)
- Present skills, education, and certificates
- Provide an easy contact form for recruiters

The style is intentionally futuristic — neon accents on a dark palette — to highlight a modern UI sensibility while remaining accessible and fast.
